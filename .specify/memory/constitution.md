# Awesome Claude Plugins

A curated list of awesome Claude marketplaces and plugins to enhance your Claude Code experience.

## Core Principles

### Comprehensive Coverage
This curated list includes all major Claude marketplaces and their plugins, ensuring developers can discover all available options in one centralized location.

### Quality Assurance
Only plugins that are verified to work with Claude Code, have clear documentation, and provide value to developers are included in this list.

### Community Driven
Contributions are encouraged through pull requests, making this list a community-maintained resource for Claude plugin discovery.

### Regular Maintenance
The list is actively maintained and updated to reflect new plugins, marketplaces, and changes in the Claude ecosystem.

### Clear Organization
Plugins are organized by marketplace with detailed descriptions, installation instructions, and usage examples for easy navigation.

2. Add the entry to the appropriate section
3. Ensure the plugin is verified and documented
4. Submit a pull request with a clear description
